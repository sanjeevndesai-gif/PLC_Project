Copa Form GUI Installer Package

How to install:
1. Extract this package if it is zipped.
2. Right-click Install-CopaFormGui.cmd and choose Run as administrator.
3. Follow the elevation prompt.
4. Launch Copa Form GUI from the desktop shortcut or Program Files\CopaFormGui\CopaFormGui.exe.

How to uninstall:
- Run Program Files\CopaFormGui\Uninstall-CopaFormGui.ps1 as administrator.
