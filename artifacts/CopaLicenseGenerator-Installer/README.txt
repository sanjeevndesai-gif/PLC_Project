Copa License Generator Installer Package

How to install:
1. Extract this package if it is zipped.
2. Right-click Install-CopaLicenseGenerator.cmd and choose Run as administrator.
3. Follow the elevation prompt.
4. Launch Copa License Generator from the desktop shortcut or Program Files\CopaLicenseGenerator\CopaLicenseGenerator.exe.

How to uninstall:
- Run Program Files\CopaLicenseGenerator\Uninstall-CopaLicenseGenerator.ps1 as administrator.
